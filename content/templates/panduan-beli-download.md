# üìÑ Panduan Membeli & Download Template

Panduan lengkap untuk membeli template, mengakses di website Using Dev, dan mendownload dari Google Drive.

## 1. Memilih & Membeli Template

### Step 1: Browsing Template
Kunjungi website Using Dev untuk melihat katalog template yang tersedia:
- **Website**: [using-dev.vercel.app](https://using-dev.vercel.app)
- Pilih kategori template yang Anda butuhkan (React, Next.js, Laravel, PHP, HTML/CSS, dll)
- Lihat preview dan deskripsi template

### Step 2: Memilih Template Pilihan
Setiap template menampilkan:
- **Preview** - Tampilan website secara keseluruhan
- **Fitur** - Feature-feature yang included
- **Tech Stack** - Teknologi yang digunakan
- **Harga** - Cost atau subscription

### Step 3: Purchase/Subscribe
- Klik tombol **"Beli" / "Purchase"** atau **"Subscribe"** untuk akses unlimited
- Login atau daftar akun Using Dev jika belum punya
- Lakukan pembayaran melalui metode yang tersedia

## 2. Mengakses Template di Using Dev

### Step 1: Login ke Akun Using Dev
1. Kunjungi [using-dev.vercel.app](https://using-dev.vercel.app)
2. Klik **"Login"** di bagian atas
3. Masukkan email dan password
4. Atau login dengan GitHub/Google (jika available)

### Step 2: Akses Dashboard
- Setelah login, masuk ke **Dashboard** atau **My Templates**
- Di sini Anda bisa melihat semua template yang sudah dibeli
- Setiap template menampilkan tombol **"Download"** atau **"View Details"**

### Step 3: Lihat Informasi Template
Klik template untuk melihat:
- **File Structure** - Struktur folder project
- **Requirements** - Node.js version, dependencies, dll
- **Setup Instructions** - Panduan instalasi
- **Download Link** - Link ke Google Drive

## 3. Download dari Google Drive

### Step 1: Klik Link Download
- Di halaman template detail, klik tombol **"Download"** atau copy link yang diberikan
- Link akan mengarahkan ke folder Google Drive
- Atau klik link Google Drive yang disediakan

### Step 2: Akses Google Drive
Pastikan Anda sudah:
- Login ke akun Google
- Akses folder template di Google Drive
- Lihat file `.zip` atau folder template

### Step 3: Download File
**Opsi A: Download Single File**
1. Klik kanan pada file template
2. Pilih **"Download"**
3. File akan ter-download ke folder Downloads Anda

**Opsi B: Download Folder (jika ada banyak file)**
1. Klik kanan pada folder
2. Pilih **"Download"**
3. Sistem akan membuat `.zip` dan mendownload

**Opsi C: Gunakan Google Drive Desktop (lebih mudah)**
1. Install [Google Drive untuk Desktop](https://www.google.com/drive/download/)
2. Sync folder ke komputer Anda
3. Akses file langsung dari komputer

### Tips Download
- **Pastikan koneksi internet stabil** - Jangan tutup browser saat download
- **Gunakan browser modern** - Chrome, Firefox, Safari, atau Edge
- **Jika file besar** - Gunakan IDM atau torrent download manager
- **Buat backup** - Download 2x jika penting atau backup ke cloud

## 4. Troubleshooting Download

### "Access Denied" di Google Drive
- Pastikan Anda sudah login ke akun Google yang benar
- Minta akses ke owner template
- Cek email untuk email konfirmasi akses

### Download Terputus
- Gunakan IDM (Internet Download Manager) untuk resume download
- Atau download lagi dengan network yang lebih stabil
- Coba gunakan VPN jika region diblokir

### File `.zip` Corrupt
- Download ulang file
- Gunakan tool seperti 7-Zip atau WinRAR untuk extract
- Coba extract dengan command line jika GUI gagal

```bash
# Linux/Mac
unzip file.zip

# Windows (PowerShell)
Expand-Archive -Path file.zip -DestinationPath ./
```

### Tidak Bisa Akses Using Dev
- Refresh halaman atau clear browser cache
- Logout dan login kembali
- Coba browser lain
- Contact support di Using Dev
